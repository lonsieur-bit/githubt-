# Registration System - User & Seller

A modern, responsive registration system with separate forms for users and sellers.

## Features

- **Dual Registration Forms**: Switch between User and Seller registration
- **Modern UI**: Beautiful gradient design with smooth animations
- **Form Validation**: Real-time validation with visual feedback
- **Password Toggle**: Show/hide password functionality
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Accessibility**: Proper labels and form structure

## How to Use

1. Simply open `index.html` in your web browser
2. Choose between "User" or "Seller" account type
3. Fill in the registration form
4. Submit to create your account

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Form handling and interactivity

## Customization

You can easily customize:
- Colors in `styles.css` (gradient colors, button colors, etc.)
- Form fields by editing the HTML structure
- Validation rules in `script.js`
- Backend integration by modifying the form submission handlers

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
